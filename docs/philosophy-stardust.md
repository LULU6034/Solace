# 浮游星尘 — 算法哲学宣言

## 核心命题

粒子不是独立个体，而是「能量场」的可见呼吸。

## 哲学论述

**一、场的不可见性与粒子的可感性。** 深邃能量视觉系统的本质悖论在于：我们试图描绘一个不可见的力场。场本身无色、无形、无质，但它作用于物质时会留下痕迹——就像风吹过水面产生的波纹。因此，每一个粒子都不是主角；它是场的信使。粒子的位置、速度、亮度、颜色，都在报告它所在空间的某个瞬间状态。

**二、吸引子作为场的锚点。** 场通过「吸引子」（attractor）将意志注入粒子群。吸引子是空间中缓慢移动的虚点，本身不可见，但每个粒子都围绕最近的吸引子做轨道运动。吸引子的位置由系统整体状态决定：音量高时彼此靠近（能量集中），平静时彼此远离（能量弥散），情绪变化时整体漂移方向改变。这种设计让 50 个独立粒子组合出统一的群体行为——观看者感觉到"有什么在指挥它们"，却看不到指挥者。

**三、音量的非线性映射。** 人的声音不是线性信号——耳语和呐喊的区别不是分贝数，而是能量的质变。因此音量对粒子的影响采用三段式映射：0-0.3 音量区间粒子做微小螺旋（仿佛在听），0.3-0.7 区间轨道速度线性增加（开始回应），0.7-1.0 区间粒子轨道半径扩大并拖尾拉长（被声音激活）。这种非线性的"聆听感"是机器随机运动无法模拟的。

**四、情绪的气候隐喻。** 情绪不是粒子的颜色——情绪是粒子所在的「气候」。neutral 是恒温层，happy 是暖流上升，sad 是冷锋过境，angry 是雷暴前夕。因此情绪变化不是瞬时切换，而是气候的缓慢漂移（1.2 秒过渡，power2.inOut 缓动）。粒子核心色反映气候温度，外围光晕始终保留青蓝基调——无论气候如何变化，深邃底色不变。

**五、鼠标作为局部扰动。** 鼠标不是吸引子——它是投入场中的一颗石子。鼠标悬停产生局部斥力场（粒子从鼠标周围散开），移动产生短暂引力尾迹，点击产生波纹脉冲（0.6 秒内向外扩散 + 再收缩回来）。这些扰动是短暂的、局部的——它不会改变场的全局状态，只是在场的表面留下涟漪。

## 视觉要义

- 粒子个体：看不见个体。看到的是群体密度、流向、亮度的整体韵律。
- 拖尾：运动的记忆。每个粒子留下它从哪里来的信息。
- 晶体闪点：场的间隙中偶尔出现的能量结晶——提醒观看者这不是液态，而是结构化能量。
- 景深：远处粒子模糊，近处粒子清晰锐利——不只是空间感，更是"意识焦点"的隐喻。

## 参数化接口

```
attractorCount: 3-5      // 吸引子数量，决定群体的组织度
orbitStrength: 0.2-1.0  // 轨道约束力，越高越有序
audioMapping: 'nonlinear' // 音量→速度的三段式映射
emotionLerp: 1.2s        // 情绪过渡时长
mouseForceType: 'repulse+ripple' // 斥力 + 波纹
```

这份哲学驱动了全部粒子行为的设计决策。每一行运动代码都可以追溯到上面的某个命题。
