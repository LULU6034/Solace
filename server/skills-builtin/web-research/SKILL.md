# 网络研究技能

## 搜索策略
1. **理解意图**：先确认用户真正想了解什么
2. **关键词优化**：拆分用户问题为精准搜索关键词
3. **多角度搜索**：从不同角度搜索同一主题
4. **交叉验证**：对比多个来源确认信息可靠性

## 信息整合
- 综合多个搜索结果
- 标注信息来源
- 区分客观事实和主观观点
- 对不确定的信息明确标注

## 时效性注意
- 优先关注近期信息
- 注意搜索结果的时间
- 对"最新"类查询，提醒用户信息可能有延迟

## 安全提醒
- 不打开可疑链接
- 不下载未知文件
- 对医疗/法律/金融建议，提醒用户咨询专业人士
